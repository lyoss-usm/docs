
# RFC / Propuesta de Proyecto: [Nombre Completo del Proyecto]

Este es nuestro primer paso antes de generar un proyecto. Ten en cuenta que el desarrollo de un proyecto no es solo programar, sino saber qué estás programando. Por eso, este es el primer paso necesario antes de colocarse a programar.

> **Instrucciones para el Jefe de Proyecto:** Por favor, completa cada sección de forma clara y concisa. Si hay detalles técnicos que aún no están definidos, indícalo como "Pendiente".

## 1. Contexto y Visión
* **Problema Actual:** [Describe el problema]
* **Objetivo Principal:** [¿Qué queremos lograr con este proyecto?]
* **Métrica de Éxito:** [¿Cómo sabremos que el proyecto funcionó? Ej: X cantidad de usuarios, reducción de tiempo en tarea Y]
* **Imagen a Proyectar:** [¿Qué sensación o cualidad debe transmitir la herramienta? Ej: Cercanía, facilidad de uso, rapidez, estilo hacker/terminal]

* **Usuarios Finales / Interesados:** [¿A quién va dirigido? Ej: Estudiantes de primer año, voluntarios, profesores, etc.]

* **Lujo / Futuro:**
  * [Escribe aquí las ideas que son geniales, pero que NO son estrictamente necesarias para la primera versión de la plataforma. Ej: Filtros avanzados 3D, integración con cámara, notificaciones push]

  * [Escribe al menos una cosa que este proyecto explícitamente NO intentará resolver. Ej: No es una red social, no tendrá interfaz gráfica (solo terminal)]
