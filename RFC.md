
# RFC / Propuesta de Proyecto: [Nombre Completo del Proyecto]

> **Instrucciones para el Jefe de Proyecto:** Este documento es el requisito mínimo para autorizar el inicio de un repositorio en la organización. Se busca claridad en la intención y la arquitectura de alto nivel antes de tocar una sola línea de código.
